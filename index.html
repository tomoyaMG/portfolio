About

建設業界で現場管理マネージャーとして勤務しながら、AIを活用したツール開発を行っています。

「現場を知っている人間がAIで作る」という強みを活かし、実際に使える業務効率化ツールを短期間で開発。中小企業や個人事業主の方が抱える「ちょっとした不便」をアプリで解決するAIコンサルタントを目指しています。

日経先物 トレードツール
Webアプリ

日経225先物のエントリー判断を支援するツール。マクロ・テクニカル・リスク管理のチェックリストで感情的なトレードを防ぎ、記録・損益計算・CSV出力まで一画面で完結。

課題：チェックなしの感情的なエントリーによるルール違反を防ぎたかった。構造化された判断フローを強制する仕組みをツール化。
HTML CSS JavaScript localStorage

Python

日経225先物の価格データから酒田五法のパターンを自動検出し、チャート上に可視化する分析ツール。

課題：チャートから酒田五法のパターンを目視で探す作業が煩雑。自動検出でシグナルを瞬時に把握できるようにした。

現場配置チェックツール（HiCheck）
Python

建設現場の作業員配置を自動でPNG画像に出力するツール。TimeTreeのスケジュールデータを取得し、毎朝の配置管理業務を自動化。

課題：毎朝の配置表作成に時間がかかっていた。自動化により数秒で完了するようになった。
